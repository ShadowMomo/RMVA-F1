Color

RGBA。每个成分使用浮点小数值（Float ）。

超类

  • Object

类方法

Color.new(red, green, blue[, alpha])
Color.new (RGSS3)

生成色彩对象。alpha 值省略时使用 255。

如果参数没有指定，则默认设定为(0, 0, 0, 0)。 (RGSS3)

方法

set(red, green, blue[, alpha])
set(color) (RGSS3)

一次设置所有属性。

第二种用法可以从参数指定的对象中复制所有色彩属性，而无需特别设定。 (RGSS3)

属性

red

红色值（0～255）。超出范围的数值会自动修正。

green

绿色值（0～255）。超出范围的数值会自动修正。

blue

蓝色值（0～255）。超出范围的数值会自动修正。

alpha

alpha （填充度）值（0～255）。超出范围的数值会自动修正。
