RPG::Troop::Member

敌方队员的数据类。

超类

  • Object 物件类

调用者

  • RPG::Troop

属性

enemy_id

敌人 ID 编号。

x

队员的 X 坐标。

y

队员的 Y 坐标。

hidden

是否勾选「半途现身」选项的复选框。

定义

class RPG::Troop::Member
  def initialize
    @enemy_id = 1
    @x = 0
    @y = 0
    @hidden = false
  end
  attr_accessor :enemy_id
  attr_accessor :x
  attr_accessor :y
  attr_accessor :hidden
end
