Input

处理遥控器/游戏杆或键盘输入信息的模块。

在RGSS3中，使用符号而非按键序号来控制数据输入。 (RGSS3)

模块方法

Input.update

更新输入信息。原则上是每一帧调用此方法一次。

Input.press?(sym) (RGSS3)

检测与被参数 sym 指定的符号相对应的按键是否被按下。

如果按键被按下，则返回 TRUE ，否则返回 FALSE 。

if Input.press?(:C)
  do_something
end
Input.trigger?(sym) (RGSS3)

检测被参数 sym 指定的符号相对应的按键是否正在被再次按下。

再次按下是指在按键尚未被完全松开的时候再次按下（即在极短时间内连续按下）。

如果按钮被再次按下，则返回 TRUE，如果不是，返回 FALSE 。

Input.repeat?(sym) (RGSS3)

检测被参数 sym 指定的符号相对应的按键是否正在被再次按下。

不同于 trigger? 函数这里的再次按下是指在短时间内第二次按下后持续一定时间后再松开。

如果按钮被再次按下，则返回 TRUE，如果不是，返回 FALSE 。

Input.dir4

判断箭头键的状态，将信息转换成特殊的 4 方向形式，并返回与数字键盘方向对应的数字（2、4、6、8）。

若无任何箭头键（或任何相对应的按键）按下，则返回 0。

Input.dir8

判断箭头键的状态，将信息转换成特殊的 8 方向形式，并返回与数字键盘方向对应的数字（1、2、3、4、6、7、8、9）。

若无任何箭头键（或任何相对应的按键）按下，则返回 0。

常量

这些被指定的名字被用于符号的名称。比如，方向键中的右键可以被表示为 :RIGHT。 (RGSS3)

DOWN LEFT RIGHT UP

分别对应方向键中的下、左、右、上。

A B C X Y Z L R

分别对应与之相对应的按键。

SHIFT CTRL ALT

直接对应键盘上面的 Shift、CTRL 和 ALT 按键。

F5 F6 F7 F8 F9

对应与之相对应的键盘上的功能键，其他的功能键被系统保留，无法被用为其他用途。
