RPG::ME

事件音乐的数据类。此类有一个利用 Audio 模块播放自身的方法。

超类

  • RPG::AudioFile

调用者

  • RPG::System
  • RPG::EventCommand

类方法

RPG::ME.stop

停止播放事件音乐。

RPG::ME.fade(time)

开始淡出事件音乐。time 是以毫秒计算的淡出时间长度。

方法

play

开始播放事件音乐。

定义

class RPG::ME < RPG::AudioFile
  def play
    if @name.empty?
      Audio.me_stop
    else
      Audio.me_play('Audio/ME/' + @name, @volume, @pitch)
    end
  end
  def self.stop
    Audio.me_stop
  end
  def self.fade(time)
    Audio.me_fade(time)
  end
end
