RPG::Animation

动画的数据类。

超类

  • Object

属性

id

动画 ID 标号。

name

动画名称。

animation1_name

胶片素材1 文件名。

animation1_hue

胶片素材1 色调调整值（0～360）。

animation2_name

胶片素材2 图文件名。

animation2_hue

胶片素材2 色调调整值（0～360）。

position

动画显示基准位置（0：头上、1：中心、2：脚下、3：画面）。

frame_max

可用帧数。

frames

图象元内容。是由 RPG::Animation::Frame 对象所组成的数组。

timings

音效和闪烁效果时间内容。是由 RPG::Animation::Timing 对象所组成的数组。

方法

to_screen?

判断该动画是否全屏显示，当 position 的值为3的时候返回true。

内部类

  • RPG::Animation::Frame
  • RPG::Animation::Timing

定义

class RPG::Animation
  def initialize
    @id = 0
    @name = ''
    @animation1_name = ''
    @animation1_hue = 0
    @animation2_name = ''
    @animation2_hue = 0
    @position = 1
    @frame_max = 1
    @frames = [RPG::Animation::Frame.new]
    @timings = []
  end
  def to_screen?
    @position == 3
  end
  attr_accessor :id
  attr_accessor :name
  attr_accessor :animation1_name
  attr_accessor :animation1_hue
  attr_accessor :animation2_name
  attr_accessor :animation2_hue
  attr_accessor :position
  attr_accessor :frame_max
  attr_accessor :frames
  attr_accessor :timings
end
