RPG::CommonEvent

公共事件的数据类。

超类

  • Object

属性

id

事件编号。

name

事件名称。

trigger

触发条件（0：正常执行，1：自动执行，2：并行处理）。

switch_id

条件开关编号。

list

事件指令集（RPG::EventCommand）。

方法

autorun?

判断事件是否是自动执行，当触发条件是 1 时返回true。

parallel?

判断事件是否是并行处理，当触发条件是 2 时返回true。

定义

class RPG::CommonEvent
  def initialize
    @id = 0
    @name = ''
    @trigger = 0
    @switch_id = 1
    @list = [RPG::EventCommand.new]
  end
  def autorun?
    @trigger == 1
  end
  def parallel?
    @trigger == 2
  end
  attr_accessor :id
  attr_accessor :name
  attr_accessor :trigger
  attr_accessor :switch_id
  attr_accessor :list
end
