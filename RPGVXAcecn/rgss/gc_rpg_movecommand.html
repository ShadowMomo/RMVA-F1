RPG::MoveCommand

移动轨迹指令的数据类。

超类

  • Object 对象类

调用者

  • RPG::MoveRoute

属性

code

移动轨迹指令代码。

parameters

包含移动轨迹指令各项参数的数组，不同移动轨迹指令有不同内容。

定义

class RPG::MoveCommand
  def initialize(code = 0, parameters = [])
    @code = code
    @parameters = parameters
  end
  attr_accessor :code
  attr_accessor :parameters
end
