RPG::System::Vehicle

交通工具的数据类。

超类

  • Object

调用者

  • RPG::System

属性

character_name

交通工具的行走图文件名。

character_index

交通工具的行走图索引（0～7）

bgm

交通工具背景音乐（RPG::BGM ）。

start_map_id

交通工具初始位置所在的地图 ID 编号。

start_x

交通工具初始位置的 X 坐标。

start_y

交通工具初始位置的 Y 坐标。

定义

class RPG::System::Vehicle
  def initialize
    @character_name = ''
    @character_index = 0
    @bgm = RPG::BGM.new
    @start_map_id = 0
    @start_x = 0
    @start_y = 0
  end
  attr_accessor :character_name
  attr_accessor :character_index
  attr_accessor :bgm
  attr_accessor :start_map_id
  attr_accessor :start_x
  attr_accessor :start_y
end
