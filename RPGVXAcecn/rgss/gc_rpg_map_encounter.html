RPG::Map::Encounter

存储遇敌设置的数据类。

超类

  • Object 对象类

调用者

  • RPG::Map

属性

troop_id

遇敌队伍编号。

weight

权重。

region_set

对应遇敌的区域ID。

定义

class RPG::Map::Encounter
  def initialize
    @troop_id = 1
    @weight = 10
    @region_set = []
  end
  attr_accessor :troop_id
  attr_accessor :weight
  attr_accessor :region_set
end
