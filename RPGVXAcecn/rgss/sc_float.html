The floating-point number class. The implementation of Float is double in C.

比较 other 与 self；若是 self 大于 other 则返回正整数、若是 self 等于 other 则返回 0、若是 self 小于 other 则返回负整数。

当数值为正无穷则返回 1；当数值为负无穷则返回 -1。否则返回 nil。浮点数除 0 必得无穷数。

inf = 1.0/0
p inf
p inf.infinite?

=> Infinity
   1

inf = -1.0/0
p inf
p inf.infinite?

=> -Infinity
   -1


nan = 0.0/0.0
p nan
p nan.nan?

=> NaN
   true
