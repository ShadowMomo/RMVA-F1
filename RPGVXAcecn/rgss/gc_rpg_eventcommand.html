RPG::EventCommand

事件指令的数据类。

超类

  • Object

调用者

  • RPG::Event::Page
  • RPG::Troop::Page
  • RPG::CommonEvent

属性

code

事件指令代码。

indent

指令缩排深度。一般为0，每次使用「条件分歧」之类的指令就会增加 1。

parameters

包含事件指令各项参数的数组，不同事件指令有不同内容。

定义

class RPG::EventCommand
  def initialize(code = 0, indent = 0, parameters = [])
    @code = code
    @indent = indent
    @parameters = parameters
  end
  attr_accessor :code
  attr_accessor :indent
  attr_accessor :parameters
end
