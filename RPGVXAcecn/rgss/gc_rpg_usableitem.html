RPG::UsableItem

技能和物品的超类。

超类

  • RPG::BaseItem

属性

scope

效果范围。

  • 0: 无
  • 1: 单个敌人
  • 2: 全体敌人
  • 3: 一个随机敌人
  • 4: 两个随机敌人
  • 5: 三个随机敌人
  • 6: 四个随机敌人
  • 7: 单个队友
  • 8: 全体队友
  • 9: 单个队友（战斗不能）
  • 10: 全体队友（战斗不能）
  • 11: 使用者
occasion

使用场合。.

  • 0: 随时可用
  • 1: 仅战斗中
  • 2: 仅菜单中
  • 3: 不能使用
speed

速度修正。

success_rate

成功几率。

repeats

连续次数。

tp_gain

获取特技值。

hit_type

命中类型。

  • 0: 必定命中
  • 1: 物理攻击
  • 2: 魔法攻击
animation_id

释放动画编号。

damage

伤害（RPG::UsableItem::Damage）。

effects

使用效果列表。形式为 RPG::UsableItem::Effect 数组。

方法

for_opponent?

判断效果范围是否对敌方队员有效，当效果范围之值为 1、2、3、4、5 或 6 时返回 true 。

for_friend?

判断效果范围是否对己方队员有效，当效果范围之值为 7、8、9、10 或 11 时返回 true 。

for_dead_friend?

判断效果范围是否对无法战斗的己方队员有效，当效果范围之值为 9或 10 时返回 true 。

for_user?

判断效果范围是否对使用者本身有效，当效果范围之值为 11 时返回 true 。

for_one?

判断效果范围是否对单体队员有效，当效果范围之值为 1、3、4、7、9 或 11 时返回 true 。

for_random?

判断效果范围是否对随机队员有效，当效果范围之值为 4、5 或 6 时返回 true 。

number_of_targets

如果技能作用是范围性的，返回圈定的敌人数量。

for_all?

判断技能作用是全体性的，当效果范围之值为2、8或10时返回true

need_selection?

判断技能作用对象是否需要选择，当效果范围之值为1、7或9时返回true。

battle_ok?

判断物品是否可以在战斗中使用，当使用场合为 0 或 1 时返回 true 。

menu_ok?

判断物品是否可以在菜单中使用，当使用场合为 0 或 2 时返回 true 。

certain?

判断命中类型是否为必定命中，当命中类型为 0 的时候返回true。

physical?

判断命中类型是否为物理攻击，当命中类型为 1 的时候返回true。

magical?

判断命中类型是否为魔法攻击，当命中类型为 2 的时候返回true。

定义

class RPG::UsableItem < RPG::BaseItem
  def initialize
    super
    @scope = 0
    @occasion = 0
    @speed = 0
    @success_rate = 100
    @repeats = 1
    @tp_gain = 0
    @hit_type = 0
    @animation_id = 0
    @damage = RPG::UsableItem::Damage.new
    @effects = []
  end
  def for_opponent?
    [1, 2, 3, 4, 5, 6].include?(@scope)
  end
  def for_friend?
    [7, 8, 9, 10, 11].include?(@scope)
  end
  def for_dead_friend?
    [9, 10].include?(@scope)
  end
  def for_user?
    @scope == 11
  end
  def for_one?
    [1, 3, 7, 9, 11].include?(@scope)
  end
  def for_random?
    [3, 4, 5, 6].include?(@scope)
  end
  def number_of_targets
    for_random? ? @scope - 2 : 0
  end
  def for_all?
    [2, 8, 10].include?(@scope)
  end
  def need_selection?
    [1, 7, 9].include?(@scope)
  end
  def battle_ok?
    [0, 1].include?(@occasion)
  end
  def menu_ok?
    [0, 2].include?(@occasion)
  end
  def certain?
    @hit_type == 0
  end
  def physical?
    @hit_type == 1
  end
  def magical?
    @hit_type == 2
  end
  attr_accessor :scope
  attr_accessor :occasion
  attr_accessor :speed
  attr_accessor :animation_id
  attr_accessor :success_rate
  attr_accessor :repeats
  attr_accessor :tp_gain
  attr_accessor :hit_type
  attr_accessor :damage
  attr_accessor :effects
end
