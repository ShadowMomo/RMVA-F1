RPG::BGM

背景音乐的数据类。此类有一个利用 Audio 模块播放自身的方法。

超类

  • RPG::AudioFile

调用者

  • RPG::Map
  • RPG::System
  • RPG::System::Vehicle
  • RPG::EventCommand

类方法

RPG::BGM.last

返回正在播放背景音乐（RPG::BGM）。若是没有播放背景音乐，则返回 nil。

还可以存储当前播放的背景音乐播放到的位置。 (RGSS3)

RPG::BGM.stop

停止播放背景音乐。

RPG::BGM.fade(time)

开始淡出背景音乐。time 是以毫秒计算的淡出时间长度。

Methods

play([pos]) (RGSS3)

开始播放背景音乐。

如果音乐文件是 ogg 或 wav 格式，可以用 pos 参数指定播放的开始时间点。 (RGSS3)

replay (RGSS3)

重放 RPG::BGM.last 所存储的背景音乐。

定义

class RPG::BGM < RPG::AudioFile
  @@last = RPG::BGM.new
  def play(pos = 0)
    if @name.empty?
      Audio.bgm_stop
      @@last = RPG::BGM.new
    else
      Audio.bgm_play('Audio/BGM/' + @name, @volume, @pitch, pos)
      @@last = self.clone
    end
  end
  def replay
    play(@pos)
  end
  def self.stop
    Audio.bgm_stop
    @@last = RPG::BGM.new
  end
  def self.fade(time)
    Audio.bgm_fade(time)
    @@last = RPG::BGM.new
  end
  def self.last
    @@last.pos = Audio.bgm_pos
    @@last
  end
  attr_accessor :pos
end
