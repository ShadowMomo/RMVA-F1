RPG::Item

物品的数据类。

超类

  • RPG::UsableItem

属性

itype_id

物品类型编号。

  • 1: 普通物品
  • 2: 贵重物品
price

价格。

consumable

消耗品开关。

方法

key_item?

判断物品是否为贵重物品，当物品类型编号为2时返回true。

定义

class RPG::Item < RPG::UsableItem
  def initialize
    super
    @scope = 7
    @itype_id = 1
    @price = 0
    @consumable = true
  end
  def key_item?
    @itype_id == 2
  end
  attr_accessor :itype_id
  attr_accessor :price
  attr_accessor :consumable
end
