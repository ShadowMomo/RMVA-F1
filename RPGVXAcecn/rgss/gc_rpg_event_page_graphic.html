RPG::Event::Page::Graphic

地图事件块事件页「图片」的数据类。

超类

  • Object 对象类

调用者

  • RPG::Event::Page

属性

tile_id

地图元件 ID 编号。当指定图像非地图元件时，此值为 0。

character_name

角色行走图的文件名。

character_index

角色行走图索引（0～7）。

direction

角色的面向方向。（2：下、4：左、6：右、8：上）。

pattern

角色的步伐画格（0～2）。

定义

class RPG::Event::Page::Graphic
  def initialize
    @tile_id = 0
    @character_name = ''
    @character_index = 0
    @direction = 2
    @pattern = 0
  end
  attr_accessor :tile_id
  attr_accessor :character_name
  attr_accessor :character_index
  attr_accessor :direction
  attr_accessor :pattern
end
