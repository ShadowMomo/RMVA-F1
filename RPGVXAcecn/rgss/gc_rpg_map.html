RPG::Map

地图数据类。

超类

  • Object 对象类

属性

display_name

地图显示名。

tileset_id

地图元件集序号。

width

地图宽度。

height

地图高度。

scroll_type

循环方式（0：不循环、1：纵向循环、2：横向循环、3：纵横循环）。

specify_battleback

指定战斗背景的开关。

battleback1_name

如果指定战斗背景选项被打开，该属性会被用来指定下部战斗图。

battleback2_name

如果指定战斗背景选项被打开，该属性会被用来指定上部战斗图。

autoplay_bgm

自动切换BGM的开关。

bgm

如果自动切换BGM选项被打开，所切换的BGM名称（RPG::BGM ）。

autoplay_bgs

自动切换BGS的开关。

bgs

如果自动切换BGS选项被打开，所切换的BGS名称（RPG::BGS ）。

disable_dashing

禁用奔跑功能的开关.

encounter_list

遇敌列表。敌人队伍 ID 编号的数组。（RPG::Map::Encounter）

encounter_step

平均遇敌的步数。

parallax_name

远景图像的文件名。

parallax_loop_x

横向循环的开关。

parallax_loop_y

纵向循环的开关。

parallax_sx

自动横向滚动的速度。

parallax_sy

自动纵向滚动的速度。

parallax_show

在地图编辑器显示远景的开关。

note

地图的备注。

data

地图数据。地图元件 ID 编号的三维数组。（Table 表格类 ）。

events

地图事件。以事件 ID 为主键， RPG::Event 实例为对应值组成的哈希表。

内部类

定义

class RPG::Map
  def initialize(width, height)
    @display_name = ''
    @tileset_id = 1
    @width = width
    @height = height
    @scroll_type = 0
    @specify_battleback = false
    @battleback_floor_name = ''
    @battleback_wall_name = ''
    @autoplay_bgm = false
    @bgm = RPG::BGM.new
    @autoplay_bgs = false
    @bgs = RPG::BGS.new('', 80)
    @disable_dashing = false
    @encounter_list = []
    @encounter_step = 30
    @parallax_name = ''
    @parallax_loop_x = false
    @parallax_loop_y = false
    @parallax_sx = 0
    @parallax_sy = 0
    @parallax_show = false
    @note = ''
    @data = Table.new(width, height, 4)
    @events = {}
  end
  attr_accessor :display_name
  attr_accessor :tileset_id
  attr_accessor :width
  attr_accessor :height
  attr_accessor :scroll_type
  attr_accessor :specify_battleback
  attr_accessor :battleback1_name
  attr_accessor :battleback2_name
  attr_accessor :autoplay_bgm
  attr_accessor :bgm
  attr_accessor :autoplay_bgs
  attr_accessor :bgs
  attr_accessor :disable_dashing
  attr_accessor :encounter_list
  attr_accessor :encounter_step
  attr_accessor :parallax_name
  attr_accessor :parallax_loop_x
  attr_accessor :parallax_loop_y
  attr_accessor :parallax_sx
  attr_accessor :parallax_sy
  attr_accessor :parallax_show
  attr_accessor :note
  attr_accessor :data
  attr_accessor :events
end
