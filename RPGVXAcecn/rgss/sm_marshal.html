Marshal 递归模块

提供将 Ruby 对象写入或读出文件（或字符串）的能力。

模块方法

Marshal.dump(obj[, port][, limit])

将 obj 对象递归写入文件。

有些对象，如 File 文件类 或 MatchData 匹配类 ，或定义了特殊方法的对象，无法写入文件中。若尝试将这类对象写入文件会抛出 TypeError（异常：类型错误） 异常。

port 指定 IO类（或其子类）的实例。此时会返回 port。省略时，dump 会将对象递归以字符串形式返回。

若指定了 limit ，则不会将递归深于 limit （预设100）层的链接对象。若 limit 为负数，则不会进行层数检查。

Marshal.load(port)

从 port 中读取递归数据（Marshal.dump所输出的字符串）并生成与原对象同样状态的对象。port 指定 IO类（或其子类）的实例。
