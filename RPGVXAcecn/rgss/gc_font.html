Font

字体的类。字体是 Bitmap 的属性。

如果游戏目录下存在一个“Fonts”文件夹，那么就算系统中没有安装相应字体，游戏中依然可以使用该文件夹中存在的字体。

超类

  • Object

类方法

Font.new([name[, size]])

生成字体对象。

Font.exist?(name)

若系统中存在指定的字体则返回 true。

属性

name

字体的名称。可以依照喜欢的顺序指定为包含多个字体名称的字符串数组。

font.name = ["Myriad", "Verdana"]

在例子中，我们为了跨平台正常显示汉字而使用了字体的英文名，若系统中不存在优先度高的「Myriad」，则会使用第二选择「Verdana推。

预设是["Verdana", "Arial", "Courier New"]。

size

字号，预设为 24 (RGSS3).

bold

粗体标记。预设为 false。

italic

斜体标记。预设为 false。

outline (RGSS3)

文字轮廓标记，预设为true。

shadow

文字阴影标记。预设为 true (RGSS3)。设为 true 时会在文字的右下方描绘黑色的文字阴影。

color

文字颜色（Color ）。也可以调整 alpha 值。预设为 (255,255,255,255)。

alpha 值也同时用来描绘文字阴影和文字轮廓 (RGSS3) 。

out_color (RGSS3)

文字轮廓颜色（Color ），预设值为(0,0,0,128)。

类属性

default_name
default_size
default_bold
default_italic
default_shadow
default_outline (RGSS3)
default_color
default_out_color (RGSS3)

每样属性的默认值，在生成字体对象时应指定其取值。

Font.default_name = ["Myriad", "Verdana"]
Font.default_size = 22
Font.default_bold = true
