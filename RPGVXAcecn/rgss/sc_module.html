include(module ... )

添加指定模块的性质（方法和常量）。返回 self。include 是用来实现替代多重继承的揉合（mix-in）

class C
  include FileTest
  include Math
end

若是重复包含同个模块时，只有第一次的 includ 有效。此外，若是执行 include 时会导致模块间循环继承，则会抛出 ArgumentError（异常：参数异常） 异常。
