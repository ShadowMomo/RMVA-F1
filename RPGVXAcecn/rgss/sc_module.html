方法

self === obj

此方法大部分是用在 case 条件句比较用的。若是 obj 与 self 之间有着 Object#kind_of? 的关联则返回 true。换句话说，case 可以用来检查类、模块之间的从属关系。

str = String.new
case str
when String     # evaluates String === str
  p true        # => true
end

私有方法

attr_accessor(name ... )

定义属性 name 的读写方法，name 是由 Symbol 或字符串所指定的。

由 attr_accessor 所定义的方法如下：

def name
  @name
end
def name=(val)
  @name = val
end
attr_reader(name ... )

定义属性 name 的读出方法，name 是由 Symbol 或字符串所指定的。

由 attr_reader 所定义的方法如下：

def name
  @name
end
attr_writer(name ... )

定义属性 name 的写入方法（name=），name 是由 Symbol 或字符串所指定的。

由 attr_writer 所定义的方法如下：

def name=(val)
  @name = val
end
include(module ... )

添加指定模块的性质（方法和常量）。返回 self。include 是用来实现替代多重继承的揉合（mix-in）

class C
  include FileTest
  include Math
end

模块的能力会插入在类继承之间。因此，在搜索方法时，包含的模块其搜索优先度会在超类之上。

若是重复包含同个模块时，只有第一次的 includ 有效。此外，若是执行 include 时会导致模块间循环继承，则会抛出 ArgumentError（异常：参数异常） 异常。
