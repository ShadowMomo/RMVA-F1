Marshal

Ruby オブジェクトをファイル (または文字列) に書き出したり、読み戻したり する機能を提供するモジュール。

モジュール関数

Marshal.dump(obj[, port][, limit])

obj を再帰的にファイルに書き出します。

File や MatchData のインスタンス、特異メソッドを定義したオブジェクトなど、ファイルに書き出せない オブジェクトも存在します。そのようなオブジェクトをファイルに書き出そうとすると 例外 TypeError が発生します。

port には IO (またはそのサブクラス) のインスタンスを指定 します。この場合、port を返します。省略した場合には dump はそのオブジェクトをダンプした文字列を返します。

limit を指定した場合、limit 段以上深くリンクした オブジェクトをダンプできません (デフォルトは 100 レベル) 。 負の limit を指定すると深さチェックを行いません。

Marshal.load(port)

port からマーシャルデータ (Marshal.dump が出力する文字列) を読み込んで、元のオブジェクトと同じ状態をもつオブジェクトを生成しま す。port は文字列か IO (またはそのサブクラス) のインスタンス を指定します。
