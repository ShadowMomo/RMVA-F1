スーパークラス

インクルードされているモジュール

  • Enumerable

メソッド

self[key]

key に関連づけられた値を返します。該当するキーが登 録されていないときには、デフォルト値 (設定されていないときには nil) を返します。

self[key]=value

key に対して value を関連づけます。value を返し ます。

clear

ハッシュの中身を空にします。self を返します。

clone
dup

レシーバと同じ内容を持つ新しいハッシュを返します。フリーズしたハッ シュの clone は同様にフリーズされたハッシュを返しますが、 dup は内容の等しいフリーズされていないハッシュを返します。

value を返します。

delete(key)
delete(key) {|key| ... }

key に対する関連を取り除きます。取り除かれた値を返しますが、 key に対応する値が存在しないときには nil を返します。

ブロックが与えられたときには key にマッチするものがな かったときに評価し、その結果を返します。

each {|key, value| ... }
each_pair {|key, value| ... }

key と value を引数としてブロックを評価します。 self を返します。

each_key {|key| ... }

key を引数としてブロックを評価します。 self を返します。

each_value {|value| ... }

valueを引数としてブロックを評価します。 self を返します。

empty?

ハッシュが空のとき真を返します。

has_key?(key)
include?(key)

ハッシュが key をキーとして持つとき真を返します。

has_value?(value)

ハッシュが value を値として持つとき真を返します。 値の一致判定は == で行われます。

index(val)

val に対応するキーを返します。対応する要素が存在しないときには nil を返します。

該当するキーが複数存在する場合、どのキーを返すかは不定です。

keys

全キーの配列を返します。

length
size

ハッシュの要素の数を返します。

values

ハッシュの全値の配列を返します。
