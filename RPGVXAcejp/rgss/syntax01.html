字句構造と式

  • 識別子
  • コメント
  • 予約語
  • 式

Ruby ではアルファベットの大文字と小文字は区別されます。識別子と一部の リテラルの途中を除いては任意の場所に空白文字やコメントを置くことができます。 改行は行が明らかに次の行に継続するときだけ空白文字として、それ以外では文の 区切りとして解釈されます。

識別子

例:

foobar
ruby_is_simple

Ruby の識別子は、英文字または '_' から始まり、英数字または '_' からなります。 長さの制限はありません。

コメント

例:

# this is a comment line

文字列中以外の # から行末までをコメントとみなします。

予約語

予約語は以下のものです。

BEGIN    class    ensure   nil      self     when
END      def      false    not      super    while
alias    defined? for      or       then     yield
and      do       if       redo     true
begin    else     in       rescue   undef
break    elsif    module   retry    unless
case     end      next     return   until

予約語はクラス名、変数名などに用いることはできません。ただし 接頭辞 $、@ が先頭についたものは予約語とはみなされません。 また、def の後やメソッド呼び出しのピリオドの後など、メソッド名で あるとはっきり分かる場所ではメソッド名として用いることができます。

式

例:

true
(1+2)*3
foo()
if test then ok else ng end

変数やリテラル、演算子式や制御構造などをすべて含めて「式」と呼びます。

これらの式を並べたものが Ruby のプログラムとなります。式と式の間はセミ コロン (;) または改行で区切ります。ただし、バックスラッシュに続く改行は文の 区切りにならず、次の行へ継続します。

式は括弧によってグルーピングすることができます。
