RPG::Event

マップイベントのデータクラス。

スーパークラス

  • Object

参照元

  • RPG::Map

属性

id

ID。

name

名前。

x

マップ X 座標。

y

マップ Y 座標。

pages

イベントページ。RPG::Event::Page の配列です。

内部クラス

  • RPG::Event::Page

定義

class RPG::Event
  def initialize(x, y)
    @id = 0
    @name = ''
    @x = x
    @y = y
    @pages = [RPG::Event::Page.new]
  end
  attr_accessor :id
  attr_accessor :name
  attr_accessor :x
  attr_accessor :y
  attr_accessor :pages
end
